# Cold Email Objection Predictor

Predict psychological objections in cold emails before you hit send. Built with cognitive bias frameworks for B2B outreach.

## What It Does

Paste your cold email draft and get:
- **Reply Likelihood Score** — Probability of getting a response
- **Predicted Objections** — Top psychological resistance triggers detected
- **Reframes** — Specific suggestions to neutralize each objection

## Cognitive Biases Detected

| Objection | Bias | Example Trigger |
|-----------|------|-----------------|
| Status Quo Resistance | Status Quo Bias | "revolutionary", "transform" |
| Time Investment Fear | Scarcity Heuristic | "quick call", "15 minutes" |
| Stranger Danger | In-group Bias | "I'm reaching out", "my name is" |
| Hidden Cost Fear | Loss Aversion | "free", "no obligation" |
| Not-For-Me Dismissal | Base Rate Neglect | "companies like yours" |
| Slippery Slope Anxiety | Commitment Escalation | "partnership", "long-term" |
| Credibility Gap | Authority Bias (Inverted) | "expert", "guaranteed" |
| TLDR Reflex | Cognitive Load Aversion | Email > 100 words |
